Investigación sobre CSS

TEMA DESCRIPCION
¿Qué es CSS? CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo que permite diseñar y dar formato a las páginas web.
Formas de incluir CSS
    Se puede incluir CSS de tres formas:
  1. en línea (atributo style en etiquetas HTML),
  2. interno (dentro de la etiqueta <style> en el <head>) y
  3. externo (en un archivo .css enlazado con <link>).
Estructura básica de CSS Una regla CSS consta de un selector y un bloque de declaración con propiedades y valores. Ejemplo:
selector { propiedad: valor; }
Selectores de clase, ID y tipo
  1. Clase: Se define con un punto, por ejemplo:
    .mi-clase { color: red; }

  2. ID: Se define con un #, por ejemplo:
    #colores { backround color blue }

  3. Tipo: Se refiere a etiquetas HTML, por ejemplo:
    p { text-align: center; }
¿Qué es el modelo de caja de CSS? El modelo de caja en CSS representa los elementos HTML como cajas rectangulares con contenido, padding, borde y margen.
