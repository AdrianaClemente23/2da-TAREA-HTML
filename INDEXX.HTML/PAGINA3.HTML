Pseudocodigo definir la suma de dos numeros Diagrama
  1. definir n1, n2, suma, como entero
  2. escribir "ingrese el primer numero "
  3. leer n1
  4. escribir "ingrese el segundo numero "
  5. leer n2
  6. suma <- n1 + n2

  2. Escribir "Ingrese un número: "
  3. Leer num
  4. Si num > 0 Entonces
  5. Escribir "El número es positivo."
  6. Sino
  7. Si num < 0 Entonces
  8. Escribir "El número es negativo."
  9. Sino
  10. Escribir "El número es cero."

  12. FinSi
